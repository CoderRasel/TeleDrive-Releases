### ⚡ Auto-Updates
TeleDrive comes with a built-in seamless auto-updater. Once you install the app for the first time, you will automatically receive all future updates silently in the background!

## 📦 About this Repository

This repository (`TeleDrive-Releases`) is strictly used as the public distribution hub and auto-update server for the TeleDrive application. 

Every time a new version is built, our continuous integration pipeline automatically publishes the latest `.exe` installers and the `latest.json` auto-update manifest here.
